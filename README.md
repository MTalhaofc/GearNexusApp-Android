
# 🚗 GEARNEXUS – Buy & Sell Used Cars

**GEARNEXUS** is the ultimate Android app for buying and selling used cars with ease. It offers a seamless experience for users to post, browse, and manage car ads with integrated authentication, custom listings, and audio call functionality — all in one app.

## ✨ Features

- 🔐 **Multi-login system** (Email / Phone)
- 🧑‍💼 **Complete Profile Handling** with Firebase Authentication
- 📋 **Custom Ads (Create, Read, Update, Delete)**
- 🎙️ **Audio Call Integration** for direct buyer–seller communication
- 🧭 Easy navigation and clean UI for better user experience

---

## 📦 Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/MTalhaofc/GearNexusApp-Android.git
    ```

2. Open the project in **Android Studio**
3. Add your **Firebase credentials**
4. Build and run on an Android device or emulator
